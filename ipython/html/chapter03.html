Linear Regression¶

Linear regression the expected value of $y$ is assumed to be a linear combination of the input variables $x_j$. In mathematical notion, if $\hat{y}$ is the predicted value. $\hat{y}(\boldsymbol \beta, \mathbf x) = \beta_0 + \beta_1 x_1 + ... + \beta_p x_p$ The vector $\boldsymbol \beta$ and $\beta_0$ as intercept_. Later we perform classification with generalized linear models, using logistic regression.

def plotbc(x, y,z):
    plt.plot(x[z==1],z[z==1],"^r", markersize=15)
    plt.plot(x[z==-1],z[z==-1],"^b", markersize=15)

    plt.plot(x[(z==-1) & (y==1)],z[(z==-1) & (y==1)],".y", markersize=20)
    plt.plot(x[(z==1) & (y==-1)],z[(z==1) & (y==-1)],".c", markersize=20)
    plt.ylim(-1.5,1.5)
